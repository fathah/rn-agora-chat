# 简要说明
C/C++语言代码。  
源码是项目的组成部分之一。  

# 使用说明
请在使用之前运行 `generate.sh` 脚本。

# 目录介绍

## 按照平台
### android
    存放android平台特有的C++代码。例如：android下的日志打印功能，使用标准C++是无法完成的。
### ios
    存放ios平台特有的C++代码。
### win
    存放win平台特有的C++代码。
### mac
    存放mac平台特有的C++代码。

## 按照语言
### cpp
    存放对接标准C++代码的部分。
### java
    存放对接jni相关的C++代码部分。
### objc
    存放对接objc相关C++混编代码部分。

## 通用部分
### common
    存放通用代码部分。定义对象、接口定义、监听器定义等。
### core
    存放扩展sdk的主要内容的代码。
